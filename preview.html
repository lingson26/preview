독서실 좌석 추첨 프로그램

엑셀 명부를 업로드하고 좌석 수를 입력하면, 중복 신청자를 정리한 뒤 당첨자와 예비 당첨자를 자동 추첨합니다.

중복 제거 추첨 시간 기록 결과 엑셀 다운로드 재추첨 좌석번호 자동 부여

추첨 설정

1 엑셀 명부 업로드

첫 번째 시트를 기준으로 불러옵니다. 엑셀 제목 행은 이름, 동, 호수 형식으로 맞춰 주세요.

2 좌석 수 입력

좌석 수만큼 당첨자를 선정하고, 나머지 인원은 예비 순번으로 정리합니다.

3 중복 제거 설정

기본 권장 기준은 이름 + 동 + 호수입니다. 같은 사람이 같은 동/호수로 중복 신청한 경우 1명만 인정됩니다.

4 좌석번호 표기

예: 좌석 1, 좌석 2처럼 1번부터 순차 배정됩니다.

사용 순서: 엑셀 업로드 → 좌석 개수 입력 → 중복 기준 선택 → 추첨하기
재추첨을 누르면 기존 결과가 새 랜덤 결과로 대체되고, 추첨 이력에 시간이 남습니다.
